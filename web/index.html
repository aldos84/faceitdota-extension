FaceItDota - AddOn

Um das FaceItDota AddOn zu installieren füge es deinem Browser hinzu über die folgenden Links:

  • Firefox (AddOn v1.5Beta)
  • Chrome & Opera (Extension v1.0Beta)

Navigiere zu den AddOn-Einstellungen und füge die Datenquelle hinzu: https://faceit.deadsec.net/tcl.json

  • Firefox
  • Chrome
  • Opera

Info: Damit das AddOn initiiert wird auf FaceIt.com, musst du im Matchroom auf eine freie Fläche klicken, dann sollten die Spieler MMR und Positionen der bekannten Spieler angezeigt werden.
